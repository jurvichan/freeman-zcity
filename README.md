# Z-City

Bu depo, `freeman-zcity` projesinin güncel kullandığı Z-City dosya ağacıdır.
Server tarafında kullanılan ana içerik burada tutulur.

## Klasör Yapısı

- `gamemodes/zcity` - Z-City gamemode dosyaları
- `lua` - Garry's Mod Lua dosyaları

## Amaç

Bu repo, server kurulumunda kullanılan Z-City içeriğini tek bir yerde toplamak için düzenlenmiştir.
Gereksiz server dosyaları dahil edilmez.

## İçerik

Repo içinde yalnızca gerekli kök klasörler tutulur:

- `gamemodes/zcity`
- `lua`
